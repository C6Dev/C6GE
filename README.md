# DirectEngine

DirectEngine A Modern C++ Game Engine Using C# For The Editor Project Manager And Scripting

# Get Started

To Build DirectEngine Is Simple

Download The Source Code
```bash
git clone --recursive https://github.com/C6Dev/DirectEngine
```

Then In The Root Of The Project In Your Terminal Simple Run

Windows
```bash
./build-windows,bat
```

Linux
```bash
./build-linux.sh
```

MacOS
```bash
./build-macos.sh
```

The Build Files Should Be In
```bash
build/bin
```
