# Get Started

To Build C6GE Is Simple

Download The Source Code
```bash
git clone --recursive https://github.com/C6Dev/C6GE
```

Then In The Root Of The Project In Your Terminal Simple Run

Windows
```bash
./build-windows,bat
```

Linux
```bash
./build-linux.sh
```

MacOS
```bash
./build-macos.sh
```

The Build Files Should Be In
```bash
build/bin
```
